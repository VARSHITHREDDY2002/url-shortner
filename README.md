# URL Shortener

A simple URL shortener built with Express and SQLite.

## Project Structure

```
src/
├── index.ts        # App entry point
├── routes/         # Express route handlers
├── services/       # Business logic
├── db/             # Database setup and queries
└── utils/          # Shared utilities
```

## Getting Started

```bash
npm install
npx tsx src/index.ts
```
